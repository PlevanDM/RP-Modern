## 🏗️ Техническая архитектура

### **Frontend Stack**
- **React 18** с TypeScript для типобезопасности
- **Vite** для быстрой разработки и сборки
- **Tailwind CSS** для современной стилизации
- **Framer Motion** для плавных анимаций
- **Radix UI** для доступных компонентов
- **Recharts** для интерактивных графиков

### **State Management**
- **Zustand** для легкого управления состоянием
- **LocalStorage** для персистентности данных
- **React Query** для кеширования API

### **Сервисы**
- **MessagesService** - управление сообщениями и чатом
- **EarningsService** - финансовые операции и комиссии
- **ComplianceService** - правовые вопросы и соглашения
- **IntegrationService** - координация всех сервисов
