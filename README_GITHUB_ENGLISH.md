# 🚀 RepairHub Pro - Professional Repair Platform

> Professional device repair platform with 10%/5% commission system

## ⚡ Quick Start

### 🖱️ For Windows:
```bash
clean-start.bat
```

### 💻 Manual Start:
```bash
npm install
npm run dev
```

Opens on `http://localhost:3002` (or another available port)

## 🎯 Main Features

### 💬 Communication System
- Advanced chat with files and proposals
- Message reactions and editing system
- Chat pinning and archiving
- Real-time notifications

### 💰 Earnings System
- 10% (standard) / 5% (premium) commissions
- Detailed income analytics
- Balance management
- Reports and statistics

### ⚖️ Legal Protection
- Terms and conditions agreements
- User verification system
- Dispute management
- Fraud protection

### 👨‍💼 Admin Panel 2026
- AI analytics and predictions
- Real-time monitoring
- User and order management
- Financial analytics
- Security system

## 🛠 Technology Stack

- **Frontend:** React 18 + TypeScript + Vite
- **Styles:** Tailwind CSS + Framer Motion
- **State:** Zustand
- **Icons:** Lucide React
- **Charts:** Recharts
- **Internationalization:** i18next

## 📁 Project Structure

```
src/
├── components/
│   ├── ui/                 # Design system
│   ├── features/           # Functional components
│   └── common/             # Common components
├── services/               # Business logic
├── store/                  # State management
├── types/                  # TypeScript types
└── utils/                  # Utilities
```
