# 🧪 Инструкции по тестированию RepairHub Pro

## 🚀 Быстрый старт

### 1. Запуск приложения
```bash
# Windows
clean-start.bat

# Или вручную
npm install
npm run dev
```

### 2. Открытие в браузере
Перейдите на `http://localhost:3001`

## 👥 Тестирование ролей

### **Клиент** (создание заказов)
1. Войдите как клиент
2. Перейдите в "Створити Заказ"
3. Нажмите "Создать новый заказ"
4. Заполните все 4 шага:
   - Выберите устройство (iPhone, MacBook, etc.)
   - Выберите тип ремонта (экран, батарея, etc.)
   - Укажите детали (срочность, бюджет, местоположение)
   - Загрузите фото/видео/документы
5. Нажмите "Создать заказ"

### **Майстер** (доска заказов)
1. Войдите как мастер
2. Перейдите в "Доска Замовлень"
3. Просмотрите доступные заказы
4. Нажмите на заказ для детального просмотра
5. Нажмите "Предложить" и заполните предложение:
   - Цена
   - Время выполнения
   - Описание работы
   - Условия
6. Отправьте предложение

### **Администратор** (панель управления)
1. Войдите как администратор
2. Перейдите в "Користувачі" - управление пользователями
3. Перейдите в "Замовлення" - мониторинг заказов
4. Перейдите в "Фінанси" - финансовая аналитика
5. Перейдите в "Налаштування" - настройки системы

## 💬 Тестирование системы сообщений

### **Расширенный чат**
1. Перейдите в "Розширений Чат"
2. Отправьте текстовое сообщение
3. Добавьте реакцию (👍, ❤️, 😂)
4. Отправьте файл (фото/видео/документ)
5. Отправьте предложение с ценой
6. Ответьте на сообщение (reply)
7. Отредактируйте сообщение
8. Удалите сообщение

## 🐛 Сообщение об ошибках

При обнаружении ошибок:

1. **Опишите проблему** подробно
2. **Укажите шаги** для воспроизведения
3. **Приложите скриншоты** если возможно
4. **Укажите браузер** и версию
5. **Приложите логи** из консоли

### **Формат отчета**
```
Проблема: [Описание]
Шаги: [1. 2. 3.]
Ожидаемый результат: [Что должно быть]
Фактический результат: [Что происходит]

Консоль: [Ошибки из F12]
```
