# 🚀 RepairHub Pro - Полнофункциональная платформа ремонта

> Профессиональная платформа ремонта с комиссией 10%/5% и топовыми функциями 2026

## ⚡ Быстрый запуск

### 🖱️ Для Windows:
```bash
clean-start.bat
```

### 💻 Ручной запуск:
```bash
npm install
npm run dev
```

Откроется на `http://localhost:3001` (или другом свободном порту)

## 🏗️ Архитектура

### **Frontend Stack**
- **React 18** с TypeScript
- **Vite** для быстрой разработки
- **Tailwind CSS** для стилизации
- **Framer Motion** для анимаций
- **Radix UI** для компонентов
- **Recharts** для графиков

### **State Management**
- **Zustand** для управления состоянием
- **LocalStorage** для персистентности
- **React Query** для кеширования

### **Сервисы**
- **MessagesService** - управление сообщениями
- **EarningsService** - финансовые операции
- **ComplianceService** - правовые вопросы
- **IntegrationService** - координация сервисов

## 🧪 Тестирование


1. Запустите `clean-start.bat`
2. Откройте `http://localhost:3001`
3. Войдите как клиент/мастер/админ
4. Протестируйте все функции

## 🔧 Настройка

### **Переменные окружения**
```env
VITE_APP_NAME=RepairHub Pro
VITE_APP_VERSION=1.0.0
VITE_API_URL=http://localhost:3001
```

### **Конфигурация комиссий**
```typescript
// src/services/earningsService.ts
const commissionConfigs = [
  {
    type: 'platform_fee',
    percentage: 10,  // Стандартная комиссия
    minAmount: 0
  },
  {
    type: 'platform_fee_premium',
    percentage: 5,   // Премиум комиссия
    minAmount: 5000
  }
];
```

## 🚀 Развертывание

### **Production Build**
```bash
npm run build
npm run serve
```

### **Docker**
```dockerfile
FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
RUN npm run build
EXPOSE 3000
CMD ["npm", "run", "serve"]
```
