# 🚀 RepairHub Pro - Professional Repair Platform

> Профессиональная платформа ремонта устройств с системой комиссий 10%/5%

## ⚡ Быстрый запуск

### 🖱️ Для Windows:
```bash
clean-start.bat
```

### 💻 Ручной запуск:
```bash
npm install
npm run dev
```

Откроется на `http://localhost:3002` (или другом свободном порту)

## 🎯 Основные функции

### 💬 Система коммуникации
- Расширенный чат с файлами и предложениями
- Система реакций и редактирования сообщений
- Закрепление и архивирование чатов
- Уведомления в реальном времени

### 💰 Система заработков
- Комиссии 10% (стандарт) / 5% (премиум)
- Детальная аналитика доходов
- Управление балансом
- Отчеты и статистика

### ⚖️ Правовая защита
- Соглашения и условия использования
- Система верификации пользователей
- Управление спорами
- Защита от мошенничества

### 👨‍💼 Админ-панель 2026
- AI аналитика и предсказания
- Мониторинг в реальном времени
- Управление пользователями и заказами
- Финансовая аналитика
- Система безопасности

## 🛠 Технологический стек

- **Frontend:** React 18 + TypeScript + Vite
- **Стили:** Tailwind CSS + Framer Motion
- **Состояние:** Zustand
- **Иконки:** Lucide React
- **Графики:** Recharts
- **Интернационализация:** i18next

## 📁 Структура проекта

```
src/
├── components/
│   ├── ui/                 # Дизайн-система
│   ├── features/           # Функциональные компоненты
│   └── common/             # Общие компоненты
├── services/               # Бизнес-логика
├── store/                  # Управление состоянием
├── types/                  # TypeScript типы
└── utils/                  # Утилиты
```
