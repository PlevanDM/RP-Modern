## 💰 УПРАВЛЕНИЕ КОМИССИЯМИ

### Что можно делать:
- ✅ Просматривать текущие настройки (10% стандартная, 5% премиум)
- ✅ Редактировать проценты комиссий
- ✅ Добавлять новые конфигурации
- ✅ Включать/выключать конфигурации
- ✅ Просматривать историю изменений
- ✅ Видеть статистику доходов

### Как тестировать:
1. Найдите панель "Управление Комиссиями"
2. Нажмите "Редактировать" у любой конфигурации
3. Измените процент (например, с 10% на 12%)
4. Сохраните - изменения применятся
5. Нажмите "Добавить конфигурацию" для создания новой

## 🛡️ СИСТЕМА СООТВЕТСТВИЯ

### Что можно делать:
- ✅ Просматривать все споры в системе
- ✅ Решать споры одним кликом
- ✅ Расследовать отчеты о мошенничестве
- ✅ Подтверждать/отклонять мошенничество
- ✅ Просматривать активность пользователей

### Как тестировать:
1. Откройте панель "Сообщения и Соответствие"
2. Переключайтесь между вкладками:
   - **Споры** - все споры с возможностью решения
   - **Мошенничество** - отчеты с кнопками "Подтвердить/Отклонить"
   - **Сообщения** - модерация
   - **Активность** - логи пользователей

## 🧪 ТЕСТИРОВАНИЕ В КОНСОЛИ

### Создайте тестовые данные:
```javascript
// Создать спор
const dispute = complianceService.createDispute(
  'order-123',
  'client-1', 
  'master-1',
  'client-1',
  'Плохое качество работы',
  'Мастер выполнил работу некачественно',
  'poor_quality',
  1000
);

// Создать отчет о мошенничестве
const fraudReport = complianceService.reportFraud(
  'user-1',
  'user-2', 
  'fake_profile',
  'Подозрительный профиль',
  ['screenshot1.jpg', 'screenshot2.jpg']
);

// Изменить комиссию
earningsService.updateCommissionConfig('platform-fee-standard', {
  percentage: 12 // Изменить с 10% на 12%
});
```

### Проверить статистику:
```javascript
// Получить все конфигурации комиссий
const configs = earningsService.getCommissionConfigs();
console.log('Комиссии:', configs);

// Получить все споры
const disputes = complianceService.getUserDisputes('all');
console.log('Споры:', disputes);

// Получить отчеты о мошенничестве
const fraudReports = complianceService.getUserFraudReport('all');
console.log('Мошенничество:', fraudReports);
```
