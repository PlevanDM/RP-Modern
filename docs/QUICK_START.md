# 🚀 QUICK START - РЕПАИР ХАБ ПРО

## ⚡ Быстрый запуск (2 минуты)

### 1️⃣ Установка зависимостей
```bash
npm install
```

### 2️⃣ Запуск dev сервера
```bash
npm run dev
```
Откроется на `http://localhost:5173`

### 3️⃣ Проверка типов (перед commit)
```bash
npm run type-check
```

### 4️⃣ Production сборка
```bash
npm run build
```

## 📊 ПРОВЕРКА ФУНКЦИЙ ЧЕРЕЗ CONSOLE

```javascript
// Импортируйте сервисы в консоль
import { messagesService } from './src/services/messagesService.js'
import { earningsService } from './src/services/earningsService.js'
import { complianceService } from './src/services/complianceService.js'

// Тест 1: Создание чата
const chat = messagesService.getOrCreateChat(
  ['user1', 'user2'],
  ['Иван', 'Петр'],
  'order-123'
)
console.log('Chat created:', chat)

// Тест 2: Отправка сообщения
const msg = messagesService.sendMessage(
  chat.id,
  'user1',
  'Иван',
  'avatar-url',
  'user2',
  'Привет, это тестовое сообщение!',
  'text'
)
console.log('Message sent:', msg)

// Тест 3: Создание заработка
const earning = earningsService.createEarning(
  'master-1',
  'order-123',
  'labor_income',
  1000,
  'Замовлення от Ivana',
  'UAH'
)
console.log('Earning created:', earning)

// Тест 4: Подтверждение заработка
earningsService.confirmEarning(earning.id)
console.log('Earning confirmed')

// Тест 5: Получить баланс мастера
const balance = earningsService.getMasterBalance('master-1')
console.log('Master balance:', balance)
```

## 🐛 ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Проблема: Сообщения не отправляются
```
✓ Решение: Проверьте, что оба пользователя имеют role (client/master/admin)
✓ Проверьте localStorage на предмет ошибок
✓ Откройте F12 → Console для error сообщений
```

### Проблема: Заработки не рассчитываются
```
✓ Решение: Убедитесь, что earningsService инициализирован
✓ Проверьте, что сумма заказа корректна
✓ Проверьте конфигурацию комиссий (10% стандартная, 5% премиум)
```

### Проблема: Соглашения не сохраняются
```
✓ Решение: Проверьте, что localStorage доступен
✓ Очистите localStorage: localStorage.clear()
✓ Перезагрузите страницу
```

### Проблема: Анимации не работают
```
✓ Решение: Проверьте, что Framer Motion установлен
✓ Проверьте, что GPU acceleration включен в браузере
✓ Попробуйте другой браузер (Chrome, Firefox)
```
